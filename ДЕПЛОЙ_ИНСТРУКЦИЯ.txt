ИНСТРУКЦИЯ ПО ДЕПЛОЮ
=====================

1. ПОДКЛЮЧИТЕСЬ К СЕРВЕРУ
   - Через веб-терминал в панели управления (Beget)
   - Или через SSH если у вас есть доступ

2. ЗАГРУЗИТЕ ФАЙЛЫ НА СЕРВЕР
   Файл: deploy_files.tar.gz (18KB)
   Загрузите через веб-FTP или панель управления в директорию /root/kino

3. ВЫПОЛНИТЕ КОМАНДЫ НА СЕРВЕРЕ:

# Перейдите в директорию проекта
cd /root/kino

# Распакуйте архив
tar -xzf deploy_files.tar.gz

# Активируйте виртуальное окружение
source venv/bin/activate

# Создайте миграции для новой модели PasswordResetToken
python manage.py makemigrations

# Примените миграции
python manage.py migrate

# Соберите статические файлы
python manage.py collectstatic --noinput

# Перезапустите сервер
systemctl restart gunicorn || systemctl restart django || (pkill -f gunicorn && nohup gunicorn kino_project.wsgi:application --bind 0.0.0.0:8000 &)

4. ПРОВЕРЬТЕ РАБОТУ:
   - Откройте http://212.67.9.187/login/
   - Должна быть ссылка "Забыли пароль?"
   - Перейдите на /password-reset/ и проверьте форму
   - Проверьте регистрацию - не должно быть возможности зарегистрироваться с одинаковым email/телефоном
